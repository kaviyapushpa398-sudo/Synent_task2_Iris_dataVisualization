# Synent_task2_Iris_dataVisualization

# 🌸 Iris Data Visualization Project

## 📌 Overview

This project focuses on **data visualization** using the Iris dataset. The goal is to explore and understand patterns in the dataset through clear and meaningful visual representations.

---

## 🎯 Objective

To visualize the dataset and identify patterns by creating different types of charts such as bar charts, histograms, and scatter plots.

---

## 🏢 Internship

This project was completed as part of an internship at **Synent Technologies**, where I gained hands-on experience in data visualization and exploratory analysis.

---

## 📂 Dataset

* **Name:** Iris Dataset
* Contains information about flower species with features like sepal length, sepal width, petal length, and petal width.

---

## 🛠️ Tasks Performed

### ✅ 1. Bar Chart

* Visualized the count of each species
* Helped understand class distribution

### ✅ 2. Histogram

* Showed distribution of numerical features
* Identified data spread and frequency patterns

### ✅ 3. Scatter Plot

* Compared features (e.g., sepal length vs petal length)
* Used color coding to distinguish species

### ✅ 4. Feature Comparison

* Analyzed relationships between different variables
* Observed clustering behavior among species

---
